# User Stories – Läslistan

## 1. Visa bokkatalog
Som användare  
vill jag kunna se en lista med böcker  
så att jag kan bläddra bland tillgängliga titlar.

## 2. Markera en bok som favorit
Som användare  
vill jag kunna markera en bok som favorit  
så att jag kan spara böcker jag gillar.

## 3. Avmarkera en favoritbok
Som användare  
vill jag kunna ta bort en bok från mina favoriter  
så att min favoritlista bara innehåller böcker jag verkligen vill ha kvar.

## 4. Lägga till en ny bok
Som användare  
vill jag kunna lägga till en ny bok i katalogen  
så att jag kan utöka listan med böcker jag själv vill ha med.
